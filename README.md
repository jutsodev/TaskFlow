# TaskFlow

Персональный помощник для управления задачами, целями и привычками. Построен на SwiftUI с эффектами Liquid Glass (iOS 26).

## Возможности

- ✅ **Задачи** — создание с этапами, приоритетами, категориями, таймером
- 🎯 **Цели на год** — долгосрочные цели с вехами и ежедневными действиями
- 🔄 **Привычки** — ежедневное отслеживание с серией дней
- ⏱ **Фокус-таймер** — обратный отсчёт и секундомер
- 📅 **Календарь** — визуальная сетка задач по дням
- 📓 **Журнал** — дневник настроения с благодарностями
- 📝 **Заметки** — идеи с тегами и закреплением
- 📊 **Статистика** — аналитика продуктивности
- 🔍 **Глобальный поиск** — по всем данным
- 🏆 **Достижения** — 12 бейджей мотивации
- 🌗 **Смена темы** — авто / светлая / тёмная

## Дизайн

- Чёрно-белый стиль
- Стандартные iOS кнопки и контролы
- Liquid Glass эффекты (`.glassEffect()`)
- Плавные spring-анимации

## Сборка

- iOS 26+ / Xcode 16+
- Swift 6.0 / SwiftUI
- GitHub Actions: автоматическая сборка при push
